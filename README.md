# 📊 Linear Regression Prediction

This is a simple Machine Learning project using Linear Regression 🤖

The project predicts 📈 sales based on advertising data such as:

- 📺 TV Ads
- 📻 Radio Ads
- 📰 Newspaper Ads

---

# 🚀 About the Project

In this project, historical advertising data is used to predict future sales 🔮

The model learns patterns from previous data and gives predictions using Machine Learning.
